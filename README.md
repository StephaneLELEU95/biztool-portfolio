# BizTool Portfolio

BizTool is a billing API project for freelancers and small teams.
This public repository is a portfolio overview for recruiters, schools, and clients.
The full production source code is maintained in a private repository.

## Scope

- Clients and products management
- Quotes and invoices workflows
- Payments tracking
- JWT authentication for business routes
- Test suite and clean architecture

## Tech Stack

- FastAPI
- SQLAlchemy 2
- Alembic
- Pydantic v2
- JWT (python-jose)
- Pytest
